## 自行初始化
    执行initial.dat
    执行php -v
    执行php\8.0\x86\php.exe execute.php   默认使用php8.0 x86
### execute.php
    判断是否已经有composer文件
    执行命令注册 php环境变量  normphp
    执行注册 composer
    执行
